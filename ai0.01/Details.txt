🎯 نمای کلی
مایا یک چت‌بات هوشمند فارسی زبان است که با استفاده از معماری Encoder-Decoder مبتنی بر LSTM پیاده‌سازی شده است. این پروژه از PyTorch برای آموزش مدل، Flask برای سرو API و یک رابط کاربری مدرن برای تعامل با کاربر استفاده می‌کند.

ویژگی‌های کلیدی
✅ مدل سبک و سریع با قابلیت اجرا روی CPU

✅ آموزش روی داده‌های کم (از ۲۰ جفت پرسش و پاسخ)

✅ API REST کامل با قابلیت اتصال به هر کلاینتی

✅ رابط کاربری زیبا و واکنش‌گرا

✅ پشتیبانی کامل از زبان فارسی

✅ مدیریت تاریخچه مکالمات

✅ قابلیت تنظیم دما و خلاقیت پاسخ‌ها

📊 جزئیات فنی
۱. توکنایزر (Tokenizer)
نوع: Word-level Tokenizer

اندازه واژگان: حداکثر ۵۰۰ توکن

توکن‌های ویژه:

<pad> (0): Padding

<unk> (1): کلمات ناشناخته

<sos> (2): شروع جمله

<eos> (3): پایان جمله

حداکثر طول: ۲۰ توکن

پیش‌پردازش: حذف علائم نگارشی، نرمال‌سازی فاصله‌ها

۲. مدل (ChatModel)
پارامتر	مقدار
Embedding Dimension	۶۴
Hidden Dimension	۱۲۸
تعداد لایه‌های LSTM	۱
دوطرفه (Bidirectional)	خیر
Dropout	۰.۲
بهینه‌ساز	Adam (lr=0.003)
تابع Loss	CrossEntropyLoss
Gradient Clipping	۱.۰
۳. API Endpoints
متد	مسیر	توضیح

۲. آموزش مدل
bash
python train.py
پارامترهای قابل تنظیم در train.py:

EPOCHS: تعداد دوره‌های آموزش (پیش‌فرض: ۳۰۰)

BATCH_SIZE: حجم دسته (پیش‌فرض: ۸)

LEARNING_RATE: نرخ یادگیری (پیش‌فرض: ۰.۰۰۳)

bash

python run.py chat

🔧 عیب‌یابی
خطاهای رایج
۱. خطای IndexError: index out of range

علت: عدم تطابق vocab_size در مدل و توکنایزر

راه حل: از tokenizer.vocab_size() برای تنظیم vocab_size استفاده کنید

۲. خطای ImportError: cannot import name

علت: نسخه ناسازگار فایل‌ها

راه حل: همه فایل‌ها را از نسخه نهایی جایگزین کنید

۳. عدم پاسخ‌دهی مدل

بررسی کنید توکنایزر در checkpoints/tokenizer.json وجود دارد

مطمئن شوید best_model.pt وجود دارد

۴. خطای CORS در مرورگر

راه حل: pip install flask-cors
